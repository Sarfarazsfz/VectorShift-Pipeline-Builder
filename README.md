\# VectorShift Pipeline Builder



A visual pipeline builder built for the VectorShift frontend + backend assessment.



\## Tech Stack

\- Frontend: React, React Flow

\- Backend: Python (DAG validation)



\## Features

\- Drag \& drop nodes

\- Pipeline connections

\- DAG validation

\- Clean professional UI



\## Run Locally



\### Frontend

cd frontend

npm install

npm start



\### Backend

cd backend

python -m venv venv

venv\\Scripts\\activate

pip install -r requirements.txt

python main.py
